# Discover Event Surfaces

Use this skill to identify where valuable product events should be instrumented.

Workflow:
- Understand the flow or change.
- Identify user intents, state transitions, and success/failure points.
- Map each to candidate events and properties.
- Prioritize the highest-value surfaces first.
