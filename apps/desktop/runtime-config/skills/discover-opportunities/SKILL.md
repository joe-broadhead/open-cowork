# Discover Opportunities

Use this skill when the user wants product opportunities surfaced from evidence.

Workflow:
- Gather signals from analytics, experiments, replay, and feedback.
- Convert them into concrete opportunities, not vague ideas.
- Prioritize by likely impact and confidence.
- Return a short, actionable ranked list.
