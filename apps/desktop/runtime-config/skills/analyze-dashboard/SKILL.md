# Analyze Dashboard

Use this skill when the user wants an executive summary or review of a dashboard.

Workflow:
- Read the dashboard structure and major charts.
- Group findings into wins, risks, anomalies, and open questions.
- Keep the summary aligned to the audience and decision context.
- Reference the most important charts explicitly.
