# Create Chart

Use this skill when the user wants a new chart in Amplitude.

Workflow:
- Clarify the question, metric, time range, filters, and breakdown.
- Reuse existing events and properties where possible.
- Build the chart and verify that it answers the stated question.
- Share the chart and explain what it measures.
