# Compare User Journeys

Use this skill when the user wants to compare two flows, cohorts, or segments.

Workflow:
- Define the two journeys or cohorts precisely.
- Compare key events, conversion, retention, and friction points.
- Highlight the biggest behavioral divergence.
- Recommend what to investigate or change next.
