# Create Dashboard

Use this skill when the user wants a dashboard assembled around a business question.

Workflow:
- Define the audience, decisions, and review cadence.
- Select or create the supporting charts.
- Organize them into logical sections.
- Deliver the dashboard with a short explanation of the structure.
