# Analyze Experiment

Use this skill for experiment readouts, ship decisions, or post-test analysis.

Workflow:
- Inspect experiment setup, goals, guardrails, and current status.
- Evaluate outcome metrics and segment-level behavior.
- Call out statistical strength, limitations, and business impact.
- End with a clear recommendation: ship, iterate, extend, or stop.
