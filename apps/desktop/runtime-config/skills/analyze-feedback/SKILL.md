# Analyze Feedback

Use this skill when the user wants customer feedback summarized into actionable patterns.

Workflow:
- Pull the relevant feedback set.
- Group into coherent themes.
- Surface frequency, urgency, and representative examples.
- Translate the themes into product recommendations.
