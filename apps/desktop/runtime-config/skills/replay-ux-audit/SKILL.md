# Replay UX Audit

Use this skill when the user wants UX friction diagnosed from actual behavior.

Workflow:
- Inspect replays or replay-backed signals.
- Identify hesitation, repeated actions, drop-offs, and dead ends.
- Connect the observed friction to funnel or engagement metrics.
- Return a prioritized UX audit summary.
