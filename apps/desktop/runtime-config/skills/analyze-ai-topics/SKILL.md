# Analyze AI Topics

Use this skill when the user asks about AI feature usage, topic trends, or AI interaction patterns.

Workflow:
- Define the AI feature, topic, or cohort.
- Pull the relevant usage or behavioral trends.
- Explain what changed, who is affected, and likely drivers.
- Recommend follow-up analysis if the pattern is weak or ambiguous.
