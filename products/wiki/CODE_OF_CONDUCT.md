# Code Of Conduct

OpenWiki uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project respectful, focused, and
welcoming for contributors with different backgrounds and levels of experience.

## Expected Behavior

- Be direct and constructive.
- Critique code, design, and behavior rather than people.
- Keep discussions relevant to the project.
- Respect security reporting boundaries.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information without permission.
- Deliberately derailing technical discussion.
- Publicly disclosing suspected vulnerabilities before maintainers can respond.

## Enforcement

Maintainers may moderate, edit, hide, or remove comments and contributions that
do not meet these expectations.

Report Code of Conduct concerns privately to the maintainers at
<joseph.broadhead.dev@gmail.com> with `[OpenWiki Conduct]` in the subject. Reports
are handled confidentially, and the reporter's identity is not shared without
their consent.
